# Epic 19 — Game Feel (Game Designer)

## Context

Epic 19 is a visual reskin. No new mechanics. Game feel changes are limited to:

- **Perceived immersion**: Underwater theme makes the whole app feel like one world
- **Consistency**: Same theme on stickers, summary, settings as on play

## Interaction Model

- Unchanged: tap, keyboard, drag (build-bridge)
- Feedback (correct/wrong) uses new color tokens
- No new animations beyond what Epic 11/18 already have

## Emotional Target

- "I'm playing in an underwater world"
- Cozy, safe, playful
- No stress from harsh contrasts or sterile white
